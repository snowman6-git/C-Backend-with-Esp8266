# 극저사양/최소환경구축 Http 통신 서버
- C++, 아두이노 연습용

포트폴리오, iot, 모바일 컨트롤 등등 트래픽이 극히 낮거나, 전력소모를 극히 낮추어야 할 때

Esp8266 Oled를 기준으로 여러 status 등등을 업데이트 할 예정, 추가로 SD카드와 리로드를 통해 html, css, js 등을 컴파일 없이 편하게 교체 할 수 있게 제작 예정
